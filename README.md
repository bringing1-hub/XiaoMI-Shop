# XiaoMI-Shop

小米商城前端仿站实训项目。

## 项目说明

本项目只做前端静态页面展示和基础 JavaScript 交互，不连接后端数据库，不做真实登录、订单和支付功能。

## 页面结构

- index.html：首页
- products.html：商品列表页
- detail.html：商品详情页
- cart.html：购物车页
- login.html：登录页
- css/style.css：公共样式
- js/main.js：公共脚本
- images/：图片素材
